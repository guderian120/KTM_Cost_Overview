High-Availability Architecture Cost Analysis

Architecture Overview

This cost analysis covers our modern cloud-native architecture:

①
Frontend: CloudFront + S3 for static asset hosting with global CDN caching
②
API Layer: API Gateway proxies requests to internal services
③
Compute: Network Load Balancer (private subnet) → ECS Fargate tasks
④
Deployments: Blue/Green deployments with CodeDeploy for zero downtime
⑤
Automation: GitHub Actions CI/CD + Terraform IaC

Detailed Service Cost Breakdown

1. CloudFront + S3 (Frontend)

CloudFront Pricing:

  • $0.085 per GB data transfer out (first 10TB)
  • $0.0075 per 10,000 HTTP/HTTPS requests
  • Free: Invalidations (first 1,000 paths/month)

S3 Pricing:

  • $0.023 per GB/month (standard storage)
  • $0.0004 per 1,000 GET requests

2. API Gateway

REST API Pricing:

  • $3.50 per million API calls
  • $0.09 per GB data transfer out
  • Additional caching costs if enabled

3. Network Load Balancer + ECS

NLB Pricing:

  • $0.0225 per NLB hour (~$16.20/month)
  • $0.006 per GB processed by NLB

ECS Fargate Pricing:

  • $0.04048 per vCPU per hour (~$29.15/month)
  • $0.004445 per GB per hour (~$3.20/month)
  • Example: 2 vCPU + 4GB task = ~$0.20/hour (~$144/month)

4. Blue/Green Deployments

CodeDeploy Pricing:

  • $0.02 per active deployment hour
  • Additional compute costs during transition
Note: Blue/Green requires running duplicate tasks during cutover (typically 1-2 hours)

5. DevOps Tooling

GitHub Actions:

  • Free: 2,000 minutes/month (Linux)
  • $0.008 per additional minute

Terraform Cloud:

  • Free: Up to 5 users (Community tier)
  • $20/user/month (Team tier)

Cost Optimization Tips

  • CloudFront: Use compression and cache policies to reduce data transfer
  • ECS: Right-size tasks and enable auto-scaling
  • Blue/Green: Schedule deployments during low-traffic periods
  • Terraform: Use remote state with S3 (costs <$1/month)
  • GitHub Actions: Clean up old workflows to stay within free tier
